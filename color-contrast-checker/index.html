📖 About WCAG Contrast Requirements

The Web Content Accessibility Guidelines (WCAG) 2.1 define minimum contrast ratios to ensure text is readable:

  • AA Normal Text: ≥ 4.5:1 — minimum for body text
  • AA Large Text: ≥ 3:1 — text ≥18pt or ≥14pt bold
  • AAA Normal Text: ≥ 7:1 — enhanced accessibility
  • AAA Large Text: ≥ 4.5:1 — enhanced for large text
